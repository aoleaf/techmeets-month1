# Month1 - ToDoリスト

## 概要

TechMeets Month1 の学習成果物です。シンプルなToDoリストアプリをHTML/CSS/JavaScriptのみで実装しています。

## フォルダ構成

```
month1/
└── week1/
    └── index.html   # ToDoリストアプリ本体
```

## 機能

- タスクの追加（テキスト入力 → 「追加」ボタン または Enter キー）
- タスクの完了切替（タスク名クリック）
- タスクの削除
- 完了件数の表示
- ダーク / ライトモード切替（設定はlocalStorageに保存）

## 使い方

`week1/index.html` をブラウザで開くだけで動作します。サーバー不要です。

## 技術スタック

- HTML5
- CSS3（CSS変数、Flexbox）
- JavaScript（Vanilla JS、localStorage）
